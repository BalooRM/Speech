# Speech

**Installation:**

Install ffmpeg

https://ffmpeg.org/download.html

Install Python libraries using os and environment-specific settings ([python -m] pip install ... [--user]). 

 - pip install numpy
 - pip install pydub
 - pip install SpeechRecognition
